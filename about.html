What is ClimaCast?

ClimaCast is a school project designed to deliver real-time weather updates in a clean, user-friendly way. It provides current conditions and forecasts based on your location or a city of your choice.

Project Purpose

This project was created as part of a web development course to practice HTML, CSS, JavaScript, APIs, and responsive design. The goal was to build a fully functional, visually appealing weather application from scratch.

Features

  • Current weather based on location or user input
  • 5-day weather forecast
  • Responsive design for mobile and desktop
  • Clear icon-based display of weather conditions

Technologies Used

ClimaCast is built using:

  • HTML5 & CSS3
  • JavaScript (ES6+)
  • OpenWeatherMap API
  • Responsive layout techniques
